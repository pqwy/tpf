Module type Tpf.Data

Packages up data-based entry points to a generic function.

type 'a q

The query type.

This is what we need at each inner type that we encounter.

type 'a r

The overall result type.

It's often the same as q, but it doesn't have to be.

The functions data[n] are variants of the same generic function, operating on the corresponding data representations:
